Sobre mí

Actualmente combino mi experiencia técnica con la enseñanza digital para crear herramientas, contenidos, cursos y soluciones accesibles para todos.

  • Desarrollo web con HTML, CSS, Bootstrap, Flask y WordPress
  • Integración de Raspberry Pi y automatización con Python
  • Configuración de entornos locales con XAMPP y WAMP Server
  • Uso de IA para educadores y ejecutivos (chatbots, asistentes y herramientas)
  • Capacitación en Google Workspace, productividad y herramientas digitales
  • Edición de video con Filmora y contenidos visuales para redes
